Definition

Bubble Sort is a simple sorting algorithm that works by repeatedly swapping the adjacent elements if they are in the wrong order. This process is repeated until the array is sorted.

Theory & Complexity Derivation

Bubble Sort compares adjacent elements and swaps them if they are in the wrong order. It keeps repeating this process until the list is sorted. Each iteration pushes the largest unsorted value to the end, just like bubbles rising to the surfaceâ€”hence the name. Start from the first element.

    5)Repeat the same process for the remaining elements (excluding the last one, which is already sorted).
    6)Continue until no more swaps are needed

Algorithm


// INPUT: An array of n elements
// OUTPUT: The array sorted in non-decreasing order

BubbleSort(arr[], n)
  for i = 0 to n-1
    for j = 0 to n-i-2
      if arr[j] > arr[j+1]
        swap arr[j] and arr[j+1]

Applications

  • Simple educational examples to demonstrate sorting

  • Real-time scenarios where simplicity is preferred over efficiency
